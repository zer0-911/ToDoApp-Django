# ToDoApp-Django

1. Jalankan migrasi database

```
python manage.py migrate
```

2. Buat superuser

```
python manage.py createsuperuser
```

3. Jalankan server

```
python manage.py runserver
```

4. Buka browser di

```
http://localhost:8000/todo
```

5. Untuk login ke admin di

```
http://localhost:8000/admin
```
